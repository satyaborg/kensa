## Recommended path

<Steps>
  <Step title="Install kensa and the eval skills">
    Use the [Quickstart](/quickstart) page to choose the right install path for your coding agent and package manager.
  </Step>
  <Step title="Run kensa doctor">
    Verify API keys, instrumentation, and your `.kensa/` scaffold before you spend time debugging a broken setup.
  </Step>
  <Step title="Ask your coding agent to evaluate the repo">
    In Codex, Cursor, Claude Code, OpenCode, or Gemini CLI, the happy path is simply: `evaluate this agent`.
  </Step>
  <Step title="Use the reference pages when you need precision">
    Once you are running evals, switch to [Scenarios](/scenarios), [Checks](/checks), and [Judge](/judge) for exact schema and behavior details.
  </Step>
</Steps>
